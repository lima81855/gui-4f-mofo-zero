# Organic Content Paths

## Quando usar
Use para planejar posts organicos, Reels, carrosseis e conteudo de aquecimento da pagina da oferta.

## Funil organico

```text
Dor percebida -> erro comum -> diagnostico -> solucao simples -> prova -> oferta
```

## Pilares de conteudo

1. Dor: o problema que a pessoa esta vivendo.
2. Erro: o que ela faz achando que ajuda.
3. Diagnostico: como identificar a causa.
4. Metodo: o mecanismo do produto.
5. Prova: demonstracoes verificaveis, amostras do produto, comentarios autorizados quando existirem.
6. Oferta: convite para comprar o kit.

## Regras

- O conteudo deve parecer util mesmo antes da compra.
- Cada post deve preparar uma objecao.
- Evitar conteudo generico de curiosidade sem ponte para oferta.
- Nao usar antes/depois, depoimentos ou comentarios como prova se nao houver origem verificavel e autorizada.
- CTA organico pode ser leve: "comente SOS", "veja o kit", "salve para testar".
