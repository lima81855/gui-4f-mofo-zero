# Criativos para No-Brain Stack

Use esta referencia quando a oferta tiver stack, bonus, order bump ou upsell.

## Principio central

O criativo pode vender a percepcao de valor mostrando o que vem dentro, mas sem virar uma lista poluida.

```text
Dor clara -> stack visual -> preco simples -> CTA
```

## Quando mostrar a stack

Mostrar stack quando:

- O produto e visual: kit, mapas, checklists, templates.
- O preco e baixo e a stack aumenta valor percebido.
- A pessoa precisa entender que nao esta comprando "um PDF qualquer".

Nao mostrar stack quando:

- O criativo ainda precisa educar a dor.
- A lista ficaria pequena demais no mobile.
- A promessa principal ainda nao esta clara.

## Layouts de criativo

### 1. Stack vertical

Visual:
- Mockups ou cards dos entregaveis empilhados.

Texto:
- "7 ferramentas para salvar sua planta"
- "Kit SOS Planta Morrendo"

Melhor para:
- Remarketing.
- Publico morno.
- Feed.

### 2. Checklist hero

Visual:
- Um checklist grande com 3 itens.

Texto:
- "Pode ou nao pode?"
- "Antes de adubar, veja isto"

Melhor para:
- Publico frio.
- Criativo educativo.

### 3. Diagnostico antes/depois

Visual:
- Sintoma da planta de um lado.
- Mapa ou acao recomendada do outro.

Texto:
- "Folha amarela nao e sempre falta de agua"

Melhor para:
- Produto de diagnostico.

### 4. Preco baixo + valor alto

Visual:
- Kit aberto com entregaveis.

Texto:
- "O kit completo por R$47"

Melhor para:
- Fundo de funil.
- Remarketing.

## Regras de order bump e upsell

- Nao vender order bump no criativo frio.
- Usar order bump como complemento no checkout.
- Upsell deve aparecer so como continuidade natural apos compra.
- Criativo principal vende o produto central.
