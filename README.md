# MOFO ZERO — Landing Page de Alta Conversão

Este repositório contém os arquivos da página de vendas de alta conversão do produto **MOFO ZERO** (Guia 4 Caminhos para Eliminar Mofo e Revitalizar Ambientes).

## 📂 Estrutura de Arquivos

* `index.html`: A página de vendas premium, responsiva, com animações suaves de scroll e otimizações de conversão.
* `mofo-zero-diagnostico-preview.png`: Imagem de prévia do diagnóstico do guia usada na página.
* `mofo-zero-guia-4-caminhos-preview.png`: Imagem de prévia da estrutura de 4 caminhos usada na página.
* `mofo-zero-guia-hero.svg` e `mofo-zero-hero-mvt.png`: Ativos visuais de mockups do produto.

## 🚀 Como Visualizar Localmente

Basta abrir o arquivo [index.html](index.html) diretamente no seu navegador de preferência.

## 🛠️ Tecnologias Utilizadas

* HTML5 Semântico
* Vanilla CSS3 (Aparência premium com modo escuro e efeitos de vidro)
* Vanilla JavaScript (Interações e animações de scroll)
* Otimizações para tráfego frio e MVT (Minimum Viable Traffic)
